Creating Bookmarks

  • HTML links can be used to create bookmarks,so that readers can jump to
    specific parts of a web page.
  • Bookmarks can be useful if a web page is very long.
  • To create a bookmark - first create the bookmark, then add a link to it.
  • When the link is clicked, the page will scroll down or up to the location
    with the bookmark.

The target attribute

  • linked page will be displayed in the current browser window.
  • target attribute specifies where to open the linked document
    1. _self - Default. Opens the document in the same window/tab as it was clicked
    2. _blank - Opens the document in a new window or tab
    3. _parent - Opens the document in the parent frame
    4. _top - Opens the document in the full body of the window
