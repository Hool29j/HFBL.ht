There is two types of lists

  1. order list
  2. unorder list
  3. Description list

Order list

starts with the <ol> tag. Each list item starts with the <li> tag.

the list iteams will be marked with numbers by default.

Unordered HTML List

starts with the <ul> tag. Each list item starts with the <li> tag.

The list items will be marked with bullets

But if you add type attribute. It will change in to others sign.

  • type="square"
  • type="circle"
  • etc.

HTML Description Lists

A description list is a list of terms, with a description of each term.

The <dl> tag defines the description list

The <dt> tag defines the term(name)
