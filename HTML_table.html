HTML Cell Padding

  • Cell padding specifies the space between the cell content and its borders.
  • By default the padding is set to 0.
  • To set the padding, use the CSS padding property:
    th, td {
    padding: 15px;
    }
  • To add padding only above the content, use the padding-top property.
  • others sides with the padding-bottom, padding-left, and padding-right properties
  • Cell spacing is the space between each cell.
  • By default the space is set to 2 pixels.
  • To change the space between table cells, use the CSS border-space property on the table element

Colspan

The colspan attribute in HTML specifies the number of columns a cell should span. It allows the single table cell to span the width of more than one cell or column. It provides
the same functionality as “merge cell” in a spreadsheet program like Excel.

Rowspan

The rowspan attribute in HTML specifies the number of rows a cell should span. That is if a row spans two rows, it means it will take up the space of two rows in that table. It allows
the single table cell to span the height of more than one cell or row.

Vertical Table header

A table header is a row at the top of a table used to label each column.

Align Attribute

The align attribute specifies the alignment of a table according to surrounding text. Normally, an HTML table will have a break before and after it. The align attribute
allows other HTML elements to wrap around the table.

Zebra striping

Zebra striping, coloring or shading alternate rows on a table—is thought to enhance the usability of a table by guiding the user's eye along a row. This allows us, specifically,
to style only the odd or even rows of a table.

Zebra stripes are tap into the tr (table row) element is CSS. Use the nth-child() selector and add background-color of your choice to all odd (or even) table rows.

Horizontal divider

A Horizontal divider can segment content horizontally. Horizontal dividers can also be used in combination with headers and icons to create different styles of dividers.
