Blockquote element

Blockquote element is used to indicate the quotation of a large section of text from another source. The Browser usually displays the content within <blockquote> tag as indented text.

Abbr

The <abbr> tag(Abbreviation) in HTML is used to define short form of an element. The <abbr> and <acronym> tags are used as shortened versions and used to represent a series of letters.

Marking <Abbr> can give informations to browers and search-engines.

Q Tag

The <q> HTML element indicates that the enclosed text is a short inline quotation.<q> tag defines a short quotation. Browers normally insert quotation marks around the quotation

Address

The <address> tag in HTML indicats the contact infromation of a person. It can also defines email address, social media, etc. <address> element is usually renders in italic.

browers will also add a line break before and after <address> element.

bdo

BDO stands for Bi-Directional Override. The HTML <bdo> tag is used to override the current text direction.

cite

The cite HTML element is used to describe a reference to a cited creative work, and must include the title of that work.

But, we must remember a person's name is not the title of the work. It is also usually renders in italic.

dfn

Specifies a term that is going to defined within the content, The nearest parent of the <dfn> tag must also contain the definition for term
