Html Block Elements

  1. Every HTML element has a default display value, depending on what type of element it is.
  2. There are two display values: block and inline.

Block-level elements

  1. It is always starts on a new line.
  2. It is always takes up the full width available (stretches out to the left and right as far as it can).
  3. It has a top and a bottom margin, whereas an inline element does not.

The <div> element is a block-level element.

Block level elements in HTML

Inline elements

  1. It does not start on a new line.
  2. It only takes up as much width as necessary.
