Core Attributes

There are four essential attributes which you can implement on almost all HTML elements:

  1. id -provides a unique identifier
  2. title -gives a recommended title for your element
  3. class -is implemented by combining an element through a stylesheet(CSS) and identifying its class element.
  4. style -specifying the rules for Cascading Style Sheet (CSS) in the element.
