Html Block Elements

  1. Every HTML element has a default display value, depending on what type of element it is.
  2. There are two display values: block and inline.

Block-level elements

  1. It is always starts on a new line.
  2. It is always takes up the full width available (stretches out to the left and right as far as it can).
  3. It has a top and a bottom margin, whereas an inline element does not.

The <div> element is a block-level element.

Block level elements in HTML

Inline elements

  1. It does not start on a new line.
  2. It only takes up as much width as necessary.

Inline elements in HTML

Inline elements are those which only occupy the space bounded by the tags defining the element, instead of breaking the flow of the content.

HTML (Hypertext Markup Language) elements historically were categorized as either "block-level" elements or "inline-level" elements. Since this is a presentational
characteristic it is nowadays specified by CSS in the Flow Layout.

"Div"

  1. It's often used as a container for other HTML elements.
  2. It has no required attributes, but style, class and id are common.
  3. When used together with CSS, the <div> element can be used to style blocks of content.

Definition and Usage

  1. The <div> defines a division or a section in an HTML document.
  2. The <div> is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
  3. The <div> is easily styled by using the class or id attribute.
  4. Any sort of content can be put inside the <div>!

By default, browsers always place a line break before and after the <div> element.

<Span>

  1. It is an inline container used to mark up a part of a text, or a part of a document.
  2. It has no required attributes, but style, class and id are common.
  3. When used together with CSS, the <span> element can be used to style parts of the text

Definition and Usage

  1. The <span> is an inline container used to mark up a part of a text, or a part of a document.
  2. The <span> is easily styled by CSS or manipulated with JavaScript using the class or id attribute.
  3. The <span> is much like the <div> element, but <div> is a block-level element and <span> is an inline element.
